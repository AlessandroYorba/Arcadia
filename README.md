:milky_way: Change Background Colors
------

###### [Dark Grey Background]()
```VimL
let g:arcadia_Sunset = 1
colorscheme arcadia
```

###### [Even Darker Grey Background]()
```VimL
let g:arcadia_Twilight = 1
colorscheme arcadia
```

###### [Almost Black Background]()
```VimL
let g:arcadia_Midnight = 1
colorscheme arcadia
```

###### [Black Background]()
```
let g:arcadia_Pitch = 1
colorscheme arcadia
```
